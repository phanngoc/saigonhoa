Add a cart image which links to your WooCommerce shopping cart and shows how many items are in the cart. A widget or shortcode with options.

== Description ==

WooCommerce Cart Image is a fantastic plugin that allows you to add a shopping cart image that links to your WooCommerce shopping cart page.

You can add the image using a shortcode or a widget - so that you can add it anywhere on your site.

You can even use your own image. You can also change the colors using shortcode attributes or widget options.

The plugin uses ajax to update the number of items that are in the cart.

Have a look at the screenshots to see how the plugin looks (though you can style it yourself).

Please see the installation instructions for how to use the shortcode.

WPML Multilanguage ready with mo and po files.

== Installation ==

To install, simply upload and activate the plugin. You'll be taken to the settings page where you can choose what features
you'd like to use.

To install manually:
1. Download zip file.
2. Unzip.
3. Upload the unzipped directory your /wp-content/plugins directory.
4. Go to the plugins page and activate the plugin.
5. Go to your widgets page and configure the widget.

You can also add the shortcode [cartimage]. These are the optional attributes:
[cartimage text_color="white" text_background_color="red" upload_woocommerce_cart_image="http://www.example.com/your_cart_image.png"]
